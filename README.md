# SpendSmart 💰 - Mobile App

> AI-powered expense tracker mobile app — automatically captures your expenses from SMS, no manual entry needed!

---

## About The Project

**SpendSmart** is the mobile frontend for the SpendSmart expense tracker system. It connects to the SpendSmart backend to display your expenses automatically captured from incoming SMS messages using AI/LLM technology.

### Features:
- 📱 Clean and intuitive mobile UI
- 🔐 JWT Authentication & secure login
- 💸 View and manage your expenses
- 📊 Expense dashboard
- 🔔 Real-time expense updates via backend

## Getting Started

### Prerequisites
- Node.js installed
- Android Studio (for Android) or Xcode (for iOS)
- React Native environment setup

### Installation

1. Clone the repo
```bash
git clone https://github.com/KritanshPatro/expense-tracker-frontend.git
```

2. Install dependencies
```bash
npm install
```

3. Start Metro
```bash
npm start
```

4. Run on Android
```bash
npm run android
```
